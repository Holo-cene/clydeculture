# BE-20: Extend RawEvent with pricing, ticketing, image, and availability fields

**Priority:** P2  
**Area:** Connectors  
**Status:** Open  
**Depends on:** BE-10

## Why this matters

BE-10 added `doorsAt` and `endAt` to `RawEvent`, but `external_events` has nine `_guess`
columns in the schema — seven of which still have no corresponding field in the connector
interface: `price_min_guess`, `price_max_guess`, `is_free_guess`, `ticket_url_guess`,
`ticket_url_label_guess`, `image_url_guess`, `availability_guess`. Ticketmaster returns
pricing tiers, ticket links, images, and event status on every event. Skiddle returns a
`sold_out` boolean and a `cancelled` boolean. Without interface fields, connector authors
bury this data inside `raw` and the normalisation pipeline has no contract to extract it.
Pricing and availability are user-facing; losing them silently is not acceptable.

**Step 1 — Extend `RawEvent` in `packages/connectors/src/connector.ts`:**

Replace the current `RawEvent` definition with:

```ts
export interface RawEvent {
  /** Stable upstream identifier — see CONNECTOR_GUIDE.md section 4. */
  externalId: string;
  /** Required — Clyde Culture is link-first. Skip the record if absent. */
  externalUrl: string;
  title: string;
  startAt?: string;             // ISO 8601
  endAt?: string;               // ISO 8601
  doorsAt?: string;             // ISO 8601; "Doors 7pm, Show 8pm"
  venueName?: string;
  eventTypeGuess?: string;
  tagsGuess?: string[];

  // Pricing — leave undefined rather than null when unavailable
  priceMinGuess?: number;       // Lowest ticket price in GBP
  priceMaxGuess?: number;       // Highest ticket price in GBP
  isFreeGuess?: boolean;        // True when the source explicitly marks the event free.
                                // Also set to true when priceMinGuess is 0.

  // Ticketing
  ticketUrlGuess?: string;      // Direct booking link (may differ from externalUrl)
  ticketUrlLabelGuess?: string; // e.g. "Book from Ticketmaster", "Book from Venue Website"

  // Media — only set when the source ToS explicitly permits storage.
  // Do NOT set for link-only sources (Resident Advisor, Instagram).
  imageUrlGuess?: string;

  // Availability — the source's raw status string, not yet mapped to canonical values.
  // Examples — Ticketmaster: 'onsale' | 'offsale' | 'cancelled' | 'postponed' | 'rescheduled'
  //            Skiddle: pass 'sold_out' if sold_out=true, 'cancelled' if cancelled=true
  // The normalisation pipeline maps these via source_type_category_map equivalent logic.
  availabilityGuess?: string;

  /** Full upstream payload, kept for debugging and re-parsing. */
  raw: unknown;
}
```

**Step 2 — Update `docs/CONNECTOR_GUIDE.md` section 3 (Implement the interface):**

Replace the `RawEvent` type definition in the guide with the expanded version above.
After the interface block, add a prose note covering:

- `imageUrlGuess` must only be set when the source's ToS explicitly permits storing
  images. For link-only sources (Resident Advisor, Instagram) leave it `undefined`.
- `availabilityGuess` carries the raw source string, not a canonical value. The
  normalisation pipeline maps it; the connector's job is only to forward it.
- When a source marks an event as free (price = £0), set both `isFreeGuess: true`
  and `priceMinGuess: 0`. Do not set `priceMinGuess: 0` without `isFreeGuess: true`
  — a £0 price in an API response can mean "price unknown" rather than "free".

**Step 3 — Add a source-field-availability table to `docs/CONNECTOR_GUIDE.md` section 3:**

After the prose note, add a quick-reference table so connector authors know which
upstream fields map to which `RawEvent` fields:

| RawEvent field | Ticketmaster | Skiddle | RSS | iCal | HTML |
|---|---|---|---|---|---|
| `priceMinGuess` | `priceRanges[0].min` | `min_entry_price` | ✗ | ✗ | If structured |
| `priceMaxGuess` | `priceRanges[0].max` | — | ✗ | ✗ | If structured |
| `isFreeGuess` | `priceRanges[0].max == 0` | `entry_price == "0.00"` | ✗ | ✗ | If flagged |
| `ticketUrlGuess` | `url` | `link` | ✗ (use externalUrl) | ✗ | If href present |
| `ticketUrlLabelGuess` | `"Book from Ticketmaster"` | `"Book from Skiddle"` | ✗ | ✗ | `"Book from Venue"` |
| `imageUrlGuess` | `images[0].url` (ToS permits) | ✗ link-only | ✗ | ✗ | ✗ link-only |
| `availabilityGuess` | `dates.status.code` | `sold_out` / `cancelled` booleans | ✗ | ✗ | If labelled |
