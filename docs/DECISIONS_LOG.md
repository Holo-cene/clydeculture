# Decisions Log

Small-scope implementation decisions that informed code choices but do not warrant
a full ADR. Append only — do not edit past entries. ADR-level decisions go in
`docs/decisions/`.

## Format

Each entry should include the date, the prompt or task that surfaced the decision,
the decision itself, the rationale, and the files affected.
